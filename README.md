## To install project: 
- git clone the project
- import as gradle project into eclipse
- run `gradle wrapper`

## To run project
- source the database create and insert sql scripts in mariadb
- build with `gradle build`
- run with `java -jar build/libs/Zoolander-0.0.1-SNAPSHOT.jar`
